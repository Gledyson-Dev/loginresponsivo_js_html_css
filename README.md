📌 Projeto: Login Responsivo

Este projeto consiste em uma interface de login moderna e totalmente responsiva,
desenvolvida utilizando HTML, CSS e JavaScript. O objetivo é oferecer uma experiência de usuário intuitiva e agradável

🚀 Funcionalidades
📱 Design totalmente responsivo
🎨 Interface moderna e estilizada
🔐 Campos de login (usuário e senha)
👁️ Interações com JavaScript (ex: manipulação de elementos)
⚡ Estrutura simples e de fácil entendimento

🛠️ Tecnologias utilizadas
HTML5 → Estrutura da aplicação
CSS3 → Estilização e responsividade
JavaScript → Interatividade e manipulação do DOM

Essas tecnologias trabalham juntas para criar interfaces dinâmicas, onde o HTML define a estrutura, o CSS cuida do design e o JavaScript adiciona interatividade

💡 Objetivo do projeto

Este projeto foi desenvolvido com foco em aprendizado e prática de conceitos fundamentais do desenvolvimento front-end, como:

Criação de layouts responsivos
Organização de código
Manipulação básica com JavaScript
Boas práticas de UI/UX
